## How to use this repository

Start from Phase 1 and move lesson by lesson. Each lesson is written to explain:

- what the command or concept does
- why it exists
- how Git uses it internally
- real-world examples
- common mistakes
- interview questions

## Goal

This repository is designed to grow into a complete Git masterclass from beginner to advanced level.
