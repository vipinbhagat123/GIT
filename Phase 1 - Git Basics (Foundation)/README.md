# Phase 1 - Git Basics (Foundation)

Welcome to Phase 1. This phase builds the foundation you need before moving into branching, collaboration, and advanced Git.

## Lessons

- [Lesson 1 - What is Git?](./Lesson%201%20-%20What%20is%20Git.md)
- Lesson 2 - Installing Git & Configuration
- Lesson 3 - Creating Your First Repository (`git init`)
- Lesson 4 - Working Directory, Staging Area & Repository (Deep Dive)

## What you will learn in this phase

- What Git is
- Why version control matters
- Git vs GitHub
- What a commit, history, and branch mean
- How Git is installed and configured
- How a repository is created
- How the three core areas of Git work together

## Recommended order

1. Read Lesson 1
2. Practice the quiz
3. Do the hands-on lab
4. Move to Lesson 2
5. Continue in order through Lesson 4
